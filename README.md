# 🌌 Mission: Personal Portfolio

Welcome to the bridge of my personal portfolio project. This site was designed and developed from scratch as a core milestone in my web development training.

## 🚀 The Concept
The design is centered around a **"Deep Nebula"** cosmic aesthetic. As a developer with a lifelong love for space, I approached this build with a **"Space Captain"** mindset—exploring complex layouts and navigating the challenges of modern CSS.

## 🛠️ Technical Specifications
To master the fundamentals of the web, this project was built using a **"No-Framework"** rule.
* **Architecture:** Written in 100% semantic **HTML5** and modern **CSS**.
* **Layout Engine:** Powered by **CSS Grid** and **Flexbox** for a fully responsive experience.
* **Design Fidelity:** Built to mirror a **12-column Figma layout** with 90%+ accuracy.
* **Visual FX:** Features custom **glassmorphism** (bubble glass) effects and neon accents.

## 📸 Personal Touches
* **Composition:** My background as a hobbyist photographer helped guide the UI/UX visual hierarchy and imagery.
* **Refactoring Philosophy:** Much like the "second-chance" stories I enjoy, I believe every piece of complex code deserves a chance to be refactored into something more elegant and efficient.
